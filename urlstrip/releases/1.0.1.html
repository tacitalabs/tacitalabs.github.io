Release notes

URLStrip 1.0.1 (Build 4)

Desktop beta update for macOS and Windows, focused on clearer desktop behavior, Tacita Labs metadata, Windows parity, and privacy-preserving update awareness.

What changed

  • Added manual app update checking using one static Tacita Labs release file.
  • Standardized desktop version display as URLStrip 1.0.1 (Build 4).
  • Migrated desktop app identifiers and data paths to com.tacitalabs.urlstrip with legacy Elwood fallback/migration handling.
  • Improved Windows tray behavior, including single tray icon behavior and Preferences close-to-tray behavior.
  • Improved Windows notification toggle handling and notification paths.
  • Expanded Windows Rules UI with rule source, match counts, custom rule handling, import/export, and reset controls.
  • Reduced durable Windows WebView2 profile artifacts by opening the Preferences WebView in InPrivate/incognito mode.
  • Documented Windows uninstall behavior for preserving data by default and deleting current Tacita app data only when requested.
