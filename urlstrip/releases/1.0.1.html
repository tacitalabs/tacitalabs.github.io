Release notes

URLStrip 1.0.1 (Build 11)

Desktop beta refresh for macOS and Windows, focused on rules-window readability fixes, build alignment across platforms, and refreshed published release metadata.

What changed

  • Windows Rules tab now keeps long category names readable instead of clipping them mid-character when the preferences window gets narrow.
  • Windows rules list now scrolls correctly on short windows instead of crushing every category row down to a few pixels of text.
  • Hovering a Rules category now shows the full category name in a tooltip.
  • Windows Preferences now enforce a 420×480px minimum size so casual resizing cannot break the layout.
  • Per-category rule counts now hide below 480px width so the category label has room to stay readable.
  • macOS and Windows desktop metadata are now aligned on URLStrip 1.0.1 (Build 11) for clean update checks against the shared release JSON.
  • macOS release asset was rebuilt, notarized, and republished alongside the updated Windows installer and checksum file.
