Release notes

URLStrip 1.0.2 (Build 12)

Desktop beta refresh for macOS and Windows, focused on pre-release hardening, privacy cleanup, and reliability fixes found during a final code review pass.

What changed

  • Improved ClearURLs rules update handling so downloaded rule data is validated before replacing the local cache.
  • Cleaned up fallback-rule rebuilding to avoid duplicate fallback regex work after rule reloads.
  • Reduced unnecessary stats/rule-state persistence during cleaning.
  • Added safeguards around chained redirect-cleaning on both macOS and Windows.
  • Tightened provider decoding and domain-exception state handling in the shared Apple engine.
  • Restricted Windows external URL opening to expected safe schemes.
  • Made Windows JSON persistence more crash-safe with temp-file writes and rename.
  • Moved Windows clipboard-monitor persistence outside the main app mutex to avoid UI stalls during disk I/O.
  • Fixed Windows query reconstruction so kept parameters are percent-encoded correctly after cleaning.
  • Aligned macOS and Windows desktop metadata on URLStrip 1.0.2 (Build 12) for clean update checks against the shared release JSON.
