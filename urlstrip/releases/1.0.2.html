Release notes

URLStrip 1.0.2 (Build 15)

Desktop beta refresh with the Windows rule update install fix, plus the backend hardening and diagnostics added during the investigation.

What changed

  • Fixed the Windows issue where clicking Install Rule Update appeared to do nothing because the install button stayed disabled after a rule check.
  • Preserved user rule-state and custom rules across Windows rule bundle installs instead of silently resetting them.
  • Added clearer Windows rule-update error reporting and a 30-second network timeout so stalled downloads fail visibly instead of hanging forever.
  • Kept hosted curated rule update plumbing so URLStrip can check, download, validate, and activate reviewed rule bundles without shipping a new app binary.
  • Maintained rollback behavior by keeping previous rules available when installs fail or need to be reversed.
  • Aligned desktop metadata on URLStrip 1.0.2 (Build 15) for clean update checks against the shared release JSON.
