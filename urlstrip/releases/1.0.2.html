Release notes

URLStrip 1.0.2 (Build 13)

Desktop beta refresh for macOS and Windows, adding hosted curated rule updates plus the release packaging and metadata cleanup needed to test them cleanly.

What changed

  • Added hosted curated rule update plumbing so URLStrip can check, download, validate, and activate reviewed rule bundles without shipping a new app binary.
  • Added Windows rule-update support and matching UI for manual checks and installs.
  • Added the first hosted URLStrip rules bundle and release packaging flow, including hashes, manifest generation, and release notes.
  • Improved downloaded-rule validation so bad bundles fail safely without replacing the active rules.
  • Preserved rollback behavior by keeping previous rules available when installs fail or need to be reversed.
  • Aligned macOS and Windows desktop metadata on URLStrip 1.0.2 (Build 13) for clean update checks against the shared release JSON.
